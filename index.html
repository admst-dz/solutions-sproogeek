Real case

            
Sproogeek 3D Configurator

            

              B2B-сайт продаж сувенирной и полиграфической продукции: ежедневники, скетчбуки,
              термосы и powerbank с кастомизацией прямо в браузере.
            

            

              React / Vite / Zustand
              Three.js + .glb модели
              FastAPI + PostgreSQL
              Node/Puppeteer renderer
              PDF-техкарты
              Docker + Traefik
            

          


          

            
            

              static preview
              spiral notebook
            

          

        


        

          

            
Задача

            

              Убрать долгие согласования макетов между клиентом, дилером и производством: клиент
              собирает продукт, видит 3D-превью и отправляет заказ без ручной переписки.
            

          

          

            
Решение

            

              В проекте связаны клиентский конфигуратор, роли пользователей, API заказов,
              headless-render PNG и генерация технологических PDF-карт для типографии.
            

          

          

            
Результат

            

              MVP с реальными продуктовыми сценариями: выбор параметров, логотипы, рендеры,
              кабинет клиента, дилерская панель и основа для масштабирования.
            

          

        

      


      

        

          

            
Architecture

            
Как устроен проект, который можно масштабировать.

            

              Ниже структура для реального кейса или будущего продукта. Заменишь названия сервисов,
              стек и поток данных на свои - секция уже готова как техническая витрина.
            

          


          

            

              01 / Client
              
Frontend

              
Витрина, конфигуратор, личный кабинет, загрузка логотипов, выбор параметров и оформление заявки.

              
  • React / Vite
  • State management
  • Responsive UI

            

            

              02 / Visual
              
3D Engine

              
Загрузка .glb моделей, материалы, цвета, брендирование и визуальное подтверждение результата.

              
  • Three.js
  • GLTF assets
  • Realtime preview

            

            

              03 / API
              
Backend

              
Пользователи, роли, продукты, заказы, цены, статусы, валидация входящих данных и безопасность.

              
  • FastAPI
  • PostgreSQL
  • JWT / OAuth

            

            

              04 / Workers
              
Render & PDF

              
Фоновые сервисы создают PNG-рендеры 3D-сцены и технологические PDF-документы для производства.

              
  • Puppeteer
  • Headless WebGL
  • PDF generator

            

            

              05 / Events
              
Data Flow

              
События заказа проходят через API, очередь задач, логирование, хранение файлов и уведомления.

              
  • Kafka / jobs
  • CSV / audit logs
  • Uploads storage

            

            

              06 / Ops
              
Infrastructure

              
Контейнеризация, reverse proxy, SSL, CI/CD и наблюдаемость для спокойного production-запуска.

              
  • Docker
  • Traefik
  • GitHub Actions
